## practice: functions in shell scripts

1. Try out all the example scripts in this chapter!

2. Write a script `function-params.sh` with a function `print_positional_params()` that prints the script name, the number of arguments and the first three positional parameters. In the main part of the script, call the function with parameters `fun1 fun2 fun3 fun4` and then copy the line that prints the parameter info from the function body. Execute the script with arguments `cli1 cli2 cli3`. Observe the output and explain what you see.
